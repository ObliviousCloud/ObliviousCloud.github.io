Summary of Analysis

In this report, we look to determine weather patterns for cities closer to the equator. Using Python to collect data from the Open Weather API to form a dataset of 500 hundred cities.


Once the dataset is organized, the datapoints were ploted on graphs using the Python module, Matplotlib. The datapoints consist of four different weather attributes; temperature, humidity, cloudiness, and wind speed. Each of these attributes are are measured with latitude using a scatter plot to determine patterns.
